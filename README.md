# 📱 SARA Móvil - Solicitud y Asignación de Citas

SARA Móvil es la primera versión de la aplicación móvil de la plataforma **SARA** (Sistema de Administracion de Recursos y Aplicaciones), desarrollada para facilitar la **solicitud y asignación de citas médicas** por parte de los usuarios, desde cualquier lugar y en cualquier momento.

## 🚀 Tecnologías utilizadas

La aplicación móvil está construida con una combinación de tecnologías modernas y eficientes:

- **Kotlin** (Jetpack Compose) — Para el desarrollo nativo en Android.
- **React** — Usado para la interfaz de usuario en versiones web y posibles futuras extensiones móviles.
- **Rust** — Empleado en módulos críticos para mejorar el rendimiento y seguridad del procesamiento de datos.
- **RESTful APIs** — La app se comunica en tiempo real con APIs desarrolladas en Django, brindando acceso seguro a la información del usuario y del sistema.

## 🔗 Conexión con APIs

La app se conecta directamente con las APIs de la plataforma SARA para:

- Autenticación de usuarios.
- Registro y asignación de citas.
- Acceso a datos personales y médicos básicos.

## 🌐 Integración con otros módulos

Actualmente, la aplicación se enfoca en el módulo de citas médicas, pero se encuentra diseñada para permitir futuras integraciones con otros componentes disponibles en la versión web del sistema, como:

- Informes a entes de control.
- Gestion de identidades.
- Interoperabilidad.
- Módulos administrativos.

## 🛠️ En desarrollo

- Módulo de recordatorios automáticos para citas.
- Integración de notificaciones push.
- Versión iOS (en evaluación).
